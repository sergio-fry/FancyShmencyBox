FancyShmancyBox
===============

This is s [FancyBox](http://fancybox.net/) fork.

## Features
* All Fancybox 1.3.4 features
* open element inside fancybox without clicking on a button

## Example
    <div id="message">Hallo!</div>

    <script type="text/javascript">
      $(function() {
        $("#message").showFancybox();
      });
    </script>
